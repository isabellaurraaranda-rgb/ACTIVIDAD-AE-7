Actividad AE-7

Módulo 2 – Fundamentos de Desarrollo Front-End

Clase 17: Uso de Git y GitHub

Descripción

En esta actividad se trabajó con Git y GitHub para crear un repositorio, registrar cambios mediante commits y sincronizar el proyecto con un repositorio remoto.

Conclusión

Git y GitHub permiten llevar un control de versiones ordenado y facilitan el trabajo colaborativo en proyectos de desarrollo web.
