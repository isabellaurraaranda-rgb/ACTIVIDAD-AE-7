# ACTIVIDAD-AE-7
ACTIVIDAD A EVALUAR 

## Módulo 2 – Fundamentos de Desarrollo Front-End  
### Clase 17 – Git y GitHub

### Objetivo
Comprender y aplicar el uso de Git y GitHub para el control de versiones, utilizando los comandos:
- git init
- git add
- git commit
- git branch
- git remote
- git push

### Desarrollo de la actividad
Se creó un repositorio local, se inicializó Git, se realizó el primer commit y se sincronizó con un repositorio remoto en GitHub.

### Conclusión
Git y GitHub permiten mantener un historial de cambios, trabajar de forma ordenada y colaborar de manera eficiente en proyectos de desarrollo web.
